🚀 GitHub Club

Learn • Build • Grow Together

🎯 Our Main Goal

Our goal is to teach students how to use GitHub and work in teams to build real projects.

⏱ Session Duration

Each session will be of 90 minutes including teaching, demo, and practice.

🤝 Learning Method

Students will achieve perfection through peer learning by working together, reviewing each other's work, and solving problems as a team.

🌟 Benefits

  • Learn real-world teamwork
  • Improve coding skills
  • Build projects
  • Gain confidence in technology
